# jar.output.dir
#
# jarファイルの出力先です。
# 省略した場合は、 このプロジェクトフォルダの下のtargetフォルダになります。
# Spigotサーバ実行フォルダ/plugins/ を指定すれば、Jarファイルコピーの手間が省けます。
#
# 例:
# jar.output.dir=../spigot_server/plugins/
# ※ ../ は、このプロジェクトフォルダの１つ上のフォルダという意味です。
#
# サンプル:
# jar.output.dir=target/
# jar.output.dir=../spigot_server/plugins/
#
